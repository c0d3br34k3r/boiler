Include tag

{\% include "foo.html" %}

The include tag is a simple means for including one template into the current output. Useful for pieces of common functionality and the like.

Examples

To use include is very simple:

foo.html

{% set someHtml %}

Hello World

{% end %} 
{{ someHtml }}

content.html

{% set someHtml %}

Blah blah blah

{\% include "foo.html" %}

Yada yada yada

{% end %} 
{{ someHtml }}

It would output the following:

{% set someHtml %}

Blah blah blah

Hello World

Yada yada yada

{% end %} 
{{ someHtml }}

Passing values

You can also pass values into the included template, like so:

foo.html

{% set someHtml %}

Hello, {\{ name }}

{% end %} 
{{ someHtml }}

content.html

{% set someHtml %}

Blah blah blah

{\% include "foo.html" name = "Dean" %}

Yada yada yada

{% end %} 
{{ someHtml }}

It would output the following:

{% set someHtml %}

Blah blah blah

Hello, Dean

Yada yada yada

{% end %} 
{{ someHtml }}
